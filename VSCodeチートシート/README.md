# VSCode チートシート

## 選択

矩形選択を使用すると、複数行の列ごとにテキストを編集できます。

| ショートカット              | 説明                                     |
| --------------------------- | ---------------------------------------- |
| `Ctrl + Shift + Alt + 矢印` | キーボードで矩形選択を水平方向に拡張する |

## ターミナル

| ショートカット     | 説明                   |
| ------------------ | ---------------------- |
| `Ctrl + Shift + @` | 新しいターミナルを開く |
